# weechat-translate

**Use Google Translate directly in Weechat**

Put the ```tr.py``` script in ```~/.weechat/python/``` and type ```/script load tr.py```.

Examples:

```
/tr ar goedemorgen
 صباح الخير
/tr nl,ar صباح الخير
Goede morgen
/tr en,ar صباح الخير
good morning
```
